## Overview

Prepared, cleansed, and modeled Nashville AirBNB market data using Python to develop actionable insights and an executive summary with recommendations on market entry, listing acquisition, and management strategies.

---

## Problem

Entering a new short-term rental market requires understanding pricing dynamics, neighborhood performance, and guest behavior patterns — data that is available but requires significant cleaning and statistical analysis to make actionable.

---

## Solution

- Cleaned Nashville AirBNB market data
- Applied Spearman correlations to identify relationships between listing attributes and performance
- Conducted T-Tests to compare segment performance
- Used association rule analysis to uncover listing feature patterns
- Applied cluster analysis to segment the market by listing type and location
- Produced an executive summary report with concrete entry and management recommendations

---

## Methods Used

| Method | Purpose |
| :--- | :--- |
| Spearman Correlation | Attribute-performance relationships |
| T-Tests | Segment comparison |
