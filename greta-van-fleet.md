## Overview

Scraped listener and concert location data using Data Miner and analyzed it in Tableau to create a multi-view dashboard. Produced verified future concert location recommendations for Greta Van Fleet based on 2024 touring data and listener distribution.

---

## Problem

Concert tour routing decisions are often made without systematic analysis of where fan bases are actually concentrated. This project aimed to provide a data-driven recommendation for future tour stops using listener and attendance data.

---

## Solution

- Scraped listener and location data using Data Miner
- Cleaned and structured data for analysis
- Built a Tableau dashboard featuring geographic maps, bar charts, and heat maps
- Identified high-density listener markets not currently served by 2024 tour routing

---

## Dashboard Views

| View | Purpose |
| :--- | :--- |
| Geographic Map | Listener density by region |
| Bar Charts | Top markets by listener count |
| Heat Maps | Concert frequency vs. listener concentration |
