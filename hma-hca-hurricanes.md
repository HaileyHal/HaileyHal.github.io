## Overview

Developed a Python and tkinter GUI program using a flight API to support hurricane disaster response planning. The tool cleans healthcare and disaster data, determines required personnel based on hurricane location and severity, identifies lowest-cost transportation options, and generates a comprehensive cost estimate.

---

## Problem

Coordinating healthcare personnel deployment during a hurricane disaster involves complex logistics: matching personnel needs to storm severity, sourcing available flights, and estimating total mobilization costs — all typically done manually and slowly.

---

## Solution

- Built a GUI application with Python's tkinter library for accessible, non-technical use
- Integrated a flight API to retrieve real-time transportation options
- Automated personnel requirement calculations based on storm location and severity data
- Generated itemized cost estimates for full deployment scenarios
